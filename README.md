A personal portfolio website showcasing data analysis and visualization projects built in Tableau.

## Tech

- Pure HTML/CSS — no frameworks, no dependencies
- Hosted via **GitHub Pages** (free)
- Tableau workbooks published to **Tableau Public**

## How to deploy

1. Fork or clone this repo
2. Update the placeholder links in `index.html`:
   - `YOUR-USERNAME` → your GitHub username
   - `your@email.com` → your email
   - Tableau Public profile URL
3. Push to GitHub
4. Go to **Settings → Pages → Source: main branch / root**
5. Your site will be live at `https://YOUR-USERNAME.github.io/tableau-portfolio`

## Updating project links

In `index.html`, find each `<a href="https://public.tableau.com" target="_blank" class="project-card ...">` and replace with your actual Tableau Public viz URLs.
